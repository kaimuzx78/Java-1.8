--------------Difference Between Anonymous Inner Class and Lambda Expression-------

Anonymous Inner Class is a class without a name. It is used to provide implementation of a class or interface only once.

Example:

Runnable r = new Runnable() {
    public void run() {
        System.out.println("Running");
    }
};





Lambda Expression :

Short Description:
Lambda Expression is a feature introduced in Java 8. It is a short way to implement a functional interface (an interface with only one abstract method).

Example:

Runnable r = () -> System.out.println("Running");




--------------------------------------------------

1.Anonymous Inner Class is a class without a name.
Lambda Expression is a function without a name.

2.Anonymous Inner Class can extend abstract class, concrete class, or implement an interface.
Lambda Expression can only implement a functional interface.

3.Anonymous Inner Class can implement interface with any number of abstract methods.
Lambda Expression can implement interface with only one abstract method.

4.Anonymous Inner Class code is longer and less readable.
Lambda Expression code is shorter and more readable.

5. Anonymous Inner Class was introduced before Java 8.
Lambda Expression was introduced in Java 8.
